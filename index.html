Project 1
Cryptoquotes
Cryptoquotes

Cryptoquotes is an application for making and solving cryptograms, short puzzles where the user tries to decode an encrypted text. In the case of the app, the text in question is a quote and its original author.


Technologies Used
  • Ruby on Rails: Backend RESTful API
  • PostgreSQL: Database
  • Javascript: Frontend
  • React: View rendering, event handling, and DOM manipulation
  • axios: API calls for user authentication as well as CRUDing quotes/solved_quotes
  • Heroku: API deployment/hosting
  • npm: Frontend deployment to Github pages
  • GitHub: Version control

Subidos
Subidos

Subidos is a webapp to upload your photos and browse photos from other users. Users can upload images, browse both their own images and those shared by other users, and edit or delete images that they own.


Technologies Used
  • Javascript: Frontend and backend
  • Express: Backend RESTful API
  • MongoDB: Database
  • Mongoose: Schema and ownership for database
  • Amazon S3: Persistent storage for images
  • Handlebars: View rendering
  • jQuery: Event handling and DOM manipulation
  • AJAX: API calls for user authentication as well as CRUDing photos
  • Heroku: API deployment/hosting
  • Grunt: Frontend deployment to Github pages
  • GitHub: Version control
